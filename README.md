Snake (Python)
--------------

### Controls

WASD/Arrows - Move
1-0 - Change Colors
R - Reset
Escape - Quit
P - Start/Stop Autoplay (Coming soon!)
L - Automatically Reset on Game Over (*Only available during Autoplay)
H - Show/Hide Prompy Display (*Only available during Autoplay)
